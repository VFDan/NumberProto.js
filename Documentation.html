NumberProto.js Documentation

Note: The documentation is being depracated in favor of the wiki. Once the transformation is complete, this page will be deleted.

NumberProto.js is a JavaScript plugin to help with numbers. It usesNumber.prototype.foo = function(bar) {}, so you can call x = x.foo(25); instead of x = foo(x, 25);, making lexical sense. If you use number literals, they must be in parenthesis, like this:(1.23).foo(4). This, however doesn't work with strings or booleans, so you must use(+"1").foo(2). It's also compatible with every major browser, including IE. For support, polyfills have been added. All functions return a value with type Number, unless explicitly said.

Number.prototype.round(decimals)

This function rounds the input given to the set number of decimal places. If not specified, decimals is set to 0. If decimals < 0, the function returns the number rounded to the nearest 10-decimals (see lines 9-10 for examples).

Examples

(1).round(2) === 1;
(20.1).round() === 20;
(20.2).round(2) === 20.2;
(-20.2).round(0) === -20;
(20.52346).round(1) === 20.5;
(20.523).round(0) === 21;
(20.535).round(2) === 20.54;
(20.995).round(2) === 21; //The round up causes regroups.
(33).round(-1) === 30; //10^1 is 10, and 33 rounded to the nearest 10 is 30.
(99).round(-1) === 100 //The round up causes regroups.
isNaN(NaN.round(2)) === true; //Same for all values of decimals.
isNaN(Number.POSITIVE_INFINITY.round(2)) === true; //Same for all values of decimals.

Number.prototype.typeOf()

This function outputs "Integer" if the number is a safe integer, "Float" if the number is a safe float, or NaN otherwise.

Examples

(0).typeOf() === "Integer";
(1).typeOf() === "Integer";
(1.0).typeOf() === "Integer";
(1.1).typeOf() === "Float";
isNaN(Number.POSITIVE_INFINITY.typeOf()) === true;
isNaN((0 / 0).typeOf()) === true;
isNaN(NaN.typeOf()) === true;

Number.prototype.isSafe()

Works the same as Number.isSafeInteger(), except this allows safe floats too. This returns a Boolean.

Examples

(-1).isSafe() === true;
(1).isSafe() === true;
(1.1).isSafe() === true; //Would be false with Number.isSafeInteger().
Number.MAX_SAFE_INTEGER.isSafe() === true;
(Number.MIN_SAFE_INTEGER - 1).isSafe() === false;
NaN.isSafe() === false;

Number.prototype.mod(n)

This function works the same as the mod operator(%), except it does proper modulus for negative numbers (more explanation here).

Examples

(0).mod(2) === 0; // Same for all values of n.
(1).mod(2) === 1;
(2).mod(2) === 0;
(3).mod(2) === 1;
(3).mod(4) === 3;
(7).mod(4) === 3;
isNaN((1).mod(0)) === true; //Mod uses division, and division by 0 returns NaN.
(-13).mod(51) === 38; //Returns -13 with -13 % 51
(-13).mod(-15) === -13;

Number.prototype.isDivisibleBy(y)

This function checks if a number is evenly divisible by y. It returns a Boolean

Examples


    (5).isDivisibleBy(2) === false
    (0).isDivisibleBy(4) === true
    (4.4).isDivisibleBy(4) === false //4.4 / 4 = 1 R0.4
    (55).isDivisibleBy(11) === true
    (-2).isDivisibleBy(2) === true
    (-4).isDivisibleBy(-2) === true
